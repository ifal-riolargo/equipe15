Sobre o treinamento

O treinamento seria baseado no framework
MVC web para desenvolvimento ágil com Java.
O treinamento consiste em como usar a ferramenta
e como ela seria útil no dia a dia da programação.

Esse framework diminui drasticamente o tempo de trabalho
que seria perdido com o código repetitivo.

O certificado de nosso curso, servirá amplamente para a carreira
de programador, assim como agregará seu conhecimento, através de métodos ágeis.
